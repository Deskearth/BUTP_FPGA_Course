1. 设计一个汽车尾灯自动控制系统
> 要求：根据汽车行驶状态自动控制汽车尾灯

      直行：尾灯不亮

      右转：右侧尾灯亮而且按秒闪烁，左侧尾灯不亮

      左转：左侧尾灯亮而且按秒闪烁，右侧尾灯不亮

      临时停车或故障：两侧尾灯同时闪烁

      倒车显示(可选)
